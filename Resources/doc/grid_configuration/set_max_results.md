Set max results
===============

## Usage

```php
<?php
...
$grid->setMaxResults($maxResults);
...
```

## Method parameters

|parameter|Type|Default value|Description|
|:--:|:--|:--|:--|:--|
|maxResults|integer|null|Max items in the grid|

## Exemple

```php
<?php
...
$grid->setMaxResults(50);
...
```
